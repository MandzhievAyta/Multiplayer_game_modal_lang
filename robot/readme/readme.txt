Учебное задание "Модельный язык" для робота, играющем на сервере в
многопользовательскую игру "Менеджмент":
http://www.stolyarov.info/books/pdf/gameserv.pdf
1) Для компиляции использовать команду: make all
2) Для запуска программы на модельном языке необходимо полученный файл(server) вызвать с двумя параметрами - номер порта
   сервера и количество игроков(запуск сервера с многопольз. игрой "Менеджмент"), для запуска робота необходимо запустить
   исполнительный файл - robot с тремя параметрами - ip сервера, номер порта, имя входного файла с командами роботу на
   модельном языке, грамматика которого описана в файле grammar.txt, дополненные функции помимо стандартных операций модельного
   языка можно посмотреть по приведенной выше ссылке[стр 43].
   Также в этой директории описанытестовые файлы, с примером кода для глупого и умного робота.(Весь файл описывает один цикл
   игры - месяц, он перезапускается в каждом месяце, но значения переменных сохраняются(что-то наподобии статических пермен.))
3) Используйте make clean для удаления объектных файлов.
