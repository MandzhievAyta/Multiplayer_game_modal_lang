Учебное задание "Модельный язык":
http://www.stolyarov.info/books/pdf/gameserv.pdf Глава 4
1) Для компиляции использовать команду: make all
2) Для запуска программы на модельном языке необходимо полученный файл(interp_modal) вызвать с одним параметром - названием
   входного файла, содержащего программу на модельном языке(грамматика языка описана в файле grammar.txt), также в этой
   директории описан тестовый файл, в котором проверяется основная функциональность языка - "testprgm2"
3) Используйте make clean для удаления объектных файлов
